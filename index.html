⬆ Push to GitHub

Sends every project file to a repository on your GitHub. The token is saved only in this browser and sent only to api.github.com — never to this course's server.

GitHub → Settings → Developer settings → Personal access tokens → Fine-grained → Generate. Repository access: All (or pick), Permissions → Contents: Read and write + Administration: Read and write (needed only to create new repos).
Created automatically if it doesn't exist yet. Pushing again to the same name just adds new commits.
